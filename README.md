# Project Manager Skill

A comprehensive project management skill for AI assistants, covering the complete project management knowledge system including planning, scheduling, risk management, agile/Scrum, and more.

## Overview

This skill transforms your AI assistant into an experienced professional project manager with PMP/ACP certification knowledge, proficient in both traditional waterfall and agile methodologies.

## Features

- ✅ **Project Planning & Initiation** - Charter development, WBS, resource planning
- ✅ **Schedule Management** - Gantt charts, critical path, milestone tracking
- ✅ **Risk Management** - Identification, assessment, mitigation strategies
- ✅ **Agile & Scrum** - Sprint planning, kanban, retrospectives
- ✅ **Communication** - Stakeholder management, status reporting
- ✅ **Quality Management** - Standards, assurance, control
- ✅ **Cost Management** - Budgeting, earned value analysis

## Installation

### For Claude Code
```bash
# Clone to your Claude Code skills directory
git clone https://github.com/xdmjun/project-manager.git ~/.claude/skills/project-manager
```

### For Other Tools (Cursor, Aider, Windsurf, etc.)
Copy the `SKILL.md` file to your tool's skills directory.

## Usage

Once installed, you can activate the skill by saying:

```
"Activate Project Manager mode and help me plan a new project"
```

Or:

```
"As a project manager, create a project charter for my e-commerce website"
```
